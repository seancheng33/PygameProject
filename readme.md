## 项目的起因
PygameProject名字听起来貌似很高大上，但其实只是一个联系pygame的项目。<br>
利用pygame制作各种类型的游戏的练习，最后形成自己的类似游戏引擎或者游戏库的一个项目。
开这个坑就是为了让自己可以能完成这样一个学习pygame的目标。<br>
每个集合在这里的小游戏，都有自己的readme文件,里面包含了项目的开发日志等。

## 预计的类别分类
