

# 五子棋游戏

#### 开发的起因
某天在微信上面看见有人分享了一个五子棋的代码，基于Python完成的，且人机对战中开发者输，<br>
故生出自己也做一个五子棋的游戏的念头,看看自己可以做到什么水平的游戏AI，写死程序，不会<br>
用到机器学习，这个暂时不收目前实现的实验点。

#### 开发环境及使用的外部库
> win10<br>
> Python 3.6.x<br>
> PyCharm Community Edition <br>
> pygame 1.9.4
